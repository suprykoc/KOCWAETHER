
### Türkçe Açıklama
**Proje Adı:** Hava Durumu Uygulaması  
**Açıklama:**  
Bu proje, Python ve Tkinter kullanılarak geliştirilmiş bir masaüstü hava durumu uygulamasıdır. OpenWeatherMap API'si üzerinden alınan verilerle güncel hava durumunu ve tahmin bilgilerini kullanıcılara sunar. Uygulama modern bir arayüze sahiptir ve hava koşullarına göre arka plan rengini dinamik olarak değiştirir.  

**Özellikler:**  
- Güncel hava durumu bilgisi (sıcaklık, açıklama, rüzgar hızı).  
- Saatlik tahmin bilgileri.  
- 5 günlük hava durumu tahmini.  
- Hava durumu koşullarına uygun emoji desteği.  
- Hava koşullarına göre dinamik arka plan renkleri.  

**Kullanım:**  
1. Uygulamayı çalıştırın.  
2. Şehir adını girin ve "Hava Durumunu Göster" butonuna tıklayın.  
3. Güncel hava durumu ve tahmin bilgileri ekranda görüntülenecektir.

### English Description
**Project Name:** Weather Application  
**Description:**  
This project is a desktop weather application developed using Python and Tkinter. It provides real-time weather updates and forecast information fetched from the OpenWeatherMap API. The application features a modern interface and dynamically changes its background color based on weather conditions.  

**Features:**  
- Current weather information (temperature, description, wind speed).  
- Hourly forecast details.  
- 5-day weather forecast.  
- Weather-specific emoji support.  
- Dynamic background colors based on weather conditions.  

**Usage:**  
1. Run the application.  
2. Enter the city name and click on the "Show Weather" button.  
3. Current weather and forecast details will be displayed on the screen.
